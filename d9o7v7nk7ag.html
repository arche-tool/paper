At every voxel face, namely \(-X,+X,-Y,+Y,-Z,+Z\), the misorientation between itself and its neighbors are calculated. This can be modeled as a graph where voxels are nodes and voxel face form edges between nodes. The edge weight is given by misorientation angle. Figure \ref{334245}-a shows a schematic representation for the two dimenaional case.
By pruning edges with higher values of misorientation values (by default this value is set to 5Â°), the graph becomes disconnected, cf. Figure \ref{334245}-c. Now the grains are simply defined as the set of voxels forming each connected subgraph network. Each grain receives a unique ID number (\(g_{id}\)) and every voxel is updated with this information, cf. Figure \ref{334245}-d.
Grain boundaries are defined as the set voxel faces where \(g_{id}^A\ne g_{id}^B\) for the any neighbouring voxel \(A\)  and \(B\). The grain boundaries can be easily find by filtering the pruned edges from the previous step (cf. Figure \ref{334245}-e) and grouping them by grain ID pairs, as described in the equation bellow:
\(b_{AB}=\{E_{WT} | W, T\, \text{are voxels}, A = g^W_{id}, B = g^T_{id}, A \neq B\}\)

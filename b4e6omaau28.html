In the ideal case, when OR holds, angular deviation \(\Delta g\) is \(0\) on Equations \ref{eq:1} and \ref{eq:2}. Therefore one can find the best \(T^{\alpha^\prime\leftrightarrow\gamma}\) value by minimizing \(\Delta g\) for the grain boundaries. In principle, only grain boundaries when OR holds (i.e. where both are derived form the same original grain) should be included in this calculation. Including other boundaries increase the error but since the number of grain boundaries exhibiting OR expected to be vastly larger than the number of prior parent grain boundaries, the noise added can ignored. This is a reasonable assumption since the martensitic phase transformation produces grain refinement and all the new grain boundaries have misorientations that holds OR.
Although the \(T^{\alpha^\prime\leftrightarrow\gamma}\) value search could be performed without grain boundary selection (at cost of increased error), a simple technique can be easily applied. By taking advantage of the prior knowledge about the theoretical values of OR (e.g. KS), one of them can be used evaluate the initial values of \(\Delta g\). Since the optimized \(T^{\alpha^\prime\leftrightarrow\gamma}\) and the theoretical OR are expected to close (a few degrees deviation) the grain boundaries where OR holds should have \(\Delta g\) values near \(0\). Therefore selecting boundaries with lower \(\Delta g\) values (current implementation defaults to \(10^\circ\)) reduces the error on the calculation in a fully automated manner.
There are two alternative methods to calculate grain boundary misorientations \(\Delta g\). One is based on the average orientation of both neighboring grains. The other one is based on the orientation in the immediate vicinity of the grain boundary, where only the first layer of voxels at each side of the grain boundary is selected. Slightly different values of OR are obtained by the two strategies due to plastic strain accommodation during phase transformation and associated orientation gradients in bulk of the grains \cite{miyamoto2009accurate}.
This new method was validated with Fe-Ni 29% allow. Figure \ref{200480} compares the distribution of angular component of \(\Delta g\) using the theoretical K-S (also used to select boundaries and initial value to minimization search) and the optimized OR, it can be noticed that the distribution shifts to the left for lower angular values, i.e. reducing the overall error, when the optimized OR is used.

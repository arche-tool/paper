Pre-processing stages: (a) the initial orientation map (b) voxel graph with edges (voxel faces) weighted by misorientation angles (c) voxel graph with pruned high-value edges (d) the set of voxels on each connected graph becomes a grain with a unique ID and a node in the microstructure graph (e) previously high-value pruned edges \(E_{AB}\) where \(g_{id}^A\ne g_{id}^B\) (f) every set of  pruned edges between nodes with same grain IDs form a grain boundary and a edge microstructure graph.

Pre-processing stages: (a) the initial orientation map (b) voxel graph with edges weighted by misorientation angles (c) voxel graph with pruned high-value edges (d) the set of voxels (nodes) on each connected graph becomes a grain with a unique ID and a node in the microstructure graph (e) select pruned edges \(E_{AB}\) where \(g_{id}^A\ne g_{id}^B\) (f) every set voxel faces between voxels with distinct grain IDs form a grain  boundary and a edge microstructure graph.
